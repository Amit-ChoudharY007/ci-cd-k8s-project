# ci-cd-k8s-project
Built a production-grade CI/CD pipeline deploying containerized applications to Kubernetes with automated rollouts and zero-downtime deployments.
# CI/CD Kubernetes Project

## Features
- Dockerized Flask app
- CI/CD with GitHub Actions
- Helm-based Kubernetes deployment
- Rolling updates (zero downtime)

## Flow
Code → Build → Push → Deploy → Kubernetes
